Projects

Here are my most recently completed personal projects.

Median

Median is a Medium application clone that allows users to publish, edit and delete stories as well as leave responses under stories written by other users. Users can also follow other authors through their profile. It uses a combination of React for the User Interface, Redux for State Management, and ES6+ JavaScript to generate a single-page experience for the user. It uses the Ruby on Rails framework for Data Management and PostgreSQL for the database. Lastly, jQuery was used to generate all the AJAX requests to the Backend.

SolarSystem

SolarSystem is a data visualization project comprised of JavaScript, other JS libraries such as D3, and calls to a third-party REST API. The web app allows users to compare planetary data amongst the eight planets of the Solar System. My motivation behind this project was to get users interested in learning more about our home planet as well as the surrounding planets that make up the Solar System. By displaying this information in a simple but visually stimulating way, I hope to garner more people's interest in my favorite natural science.
